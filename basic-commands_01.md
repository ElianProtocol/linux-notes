# Глава 1: Первое знакомство с Linux

## Что такое Linux

**Linux** — это ядро операционной системы.  
Для повседневной работы используется **дистрибутив** — ядро + набор программ и утилит.

В курсе используется **Debian 13 (trixie)** — стабильный и широко распространённый дистрибутив.

> Linux работает на серверах, Android, суперкомпьютерах и встроенных системах.

## Подключение по SSH

**SSH (Secure Shell)** — протокол безопасного удалённого доступа.

### Из Windows (встроенный SSH-клиент)

ssh student@192.168.1.100

    student — имя пользователя

    192.168.1.100 — IP-адрес сервера

При первом подключении подтвердите ключ хоста:
    Are you sure you want to continue connecting (yes/no/[fingerprint])?
Введите yes.

    При вводе пароля символы не отображаются — это нормально.

Завершение сеанса

exit

или Ctrl + D

Структура командной строки
text

student@debian:~$

Элемент	Значение
student	текущий пользователь
debian	имя хоста
~	текущий каталог (~ = домашний каталог)
$	признак обычного пользователя (# — root)


Основные команды
Команда	Назначение	Пример
whoami	имя текущего пользователя	whoami → student
hostname	имя компьютера	hostname → debian
pwd	текущий каталог	pwd → /home/student
ls	содержимое каталога	ls / — корневые каталоги
cd	смена каталога	cd /tmp, cd (возврат домой)
clear	очистка экрана	clear
exit	выход из сеанса	exit
